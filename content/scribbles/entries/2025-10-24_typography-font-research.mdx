# Elite Typography Font Collection

Research notes from migrating to baseline grid system.

## Sans-Serif Fonts

### ⭐ Inter Variable (CHOSEN)
- **What:** Neo-grotesque with optical sizing
- **Metrics:** Cap 1490, x-height 1118-1056 (variable)
- **Axes:** Weight (100-900), Optical Size (14-32)
- **Used by:** GitHub, Stripe, Vercel, Linear, Figma
- **License:** Free (SIL OFL)
- **Why:** Industry standard, perfect metrics, optical sizing

### Suisse Int'l
- **What:** Swiss grotesque, ultra-refined
- **Used by:** Apple, high-end agencies
- **License:** ~$200-400 web
- **Why:** Swiss design perfection, neutral authority

### ABC Favorit
- **What:** Modern grotesque, warmth + precision
- **Used by:** Airbnb, tech startups
- **License:** ~$300 web
- **Why:** Approachable yet professional

### Neue Haas Grotesk
- **What:** Original Helvetica, digitally refined
- **License:** Commercial
- **Why:** Timeless Swiss design standard

## Serif Fonts

### ⭐ Newsreader Variable (CHOSEN)
- **What:** Humanist serif for screen reading
- **Axes:** Weight (200-800), Optical Size (6-72pt)
- **Used by:** Editorial sites, long-form content
- **License:** Free (SIL OFL)
- **Why:** Designed for continuous reading, optical sizing

### Literata
- **What:** Google's serif for Play Books
- **License:** Free (SIL OFL)
- **Why:** Optimized for long reading sessions

### Lyon Text
- **What:** Editorial serif, refined details
- **Used by:** NY Times variants, magazines
- **License:** ~$200-500 web
- **Why:** Peak editorial typography

### Tiempos Text
- **What:** High-contrast editorial serif
- **Used by:** Bloomberg, premium publications
- **License:** ~$300-600 web
- **Why:** Distinctive, authoritative

### Source Serif Variable
- **What:** Adobe's open-source serif
- **License:** Free (SIL OFL)
- **Why:** 6 weights, 5 optical sizes, excellent metrics

### Financier
- **What:** Economic/financial editorial serif
- **License:** Commercial
- **Why:** Compact, readable, professional

## Monospace Fonts

### ⭐ JetBrains Mono (CHOSEN)
- **What:** Developer-focused with ligatures
- **Features:** 142 ligatures, 8 weights, 152 languages
- **License:** Free (Apache 2.0)
- **Why:** Excellent readability, free, ligatures

### Berkeley Mono
- **What:** Humanist monospace, variable font
- **Axes:** Width, Weight, Slant
- **Features:** 150+ ligatures, 5 widths, 12 weights
- **License:** ~$75 developer license
- **Why:** Exceptional quality, humanist warmth

### Monaspace
- **What:** Superfamily of 5 monospace fonts
- **Innovation:** "Texture healing" for monospace
- **Variants:** Neon, Argon, Krypton, Xenon, Radon
- **License:** Free (GitHub)
- **Why:** Groundbreaking variable system

### Fira Code
- **What:** Monospace with programming ligatures
- **License:** Free (SIL OFL)
- **Why:** Popular, ligatures, well-tested

## Font Pairing Philosophy

**Chosen Stack:**
- Sans: Inter Variable (geometric, technical)
- Serif: Newsreader Variable (humanist, warm)
- Mono: JetBrains Mono (humanist, functional)

**Why this works:**
- Geometric + Humanist = balanced contrast
- All variable fonts = optical sizing consistency
- All free = unlimited iteration
- Perfect metrics = baseline grid precision
